Vigenere: a simple cipher implementation in C++

Now supports command line arguments!

-e or -d to specify encryption/decryption

-if to specify input file (will append .txt if not present)
-of to specity output file (will append .txt if not present)

-key to specify key (alphabetic charecters only)

This code compiles on the x64 VC2010 compiler, other compilers not tested.

Code::Blocks 12 project files included, with targets for x86 and x64 VC2010, debug and release.
